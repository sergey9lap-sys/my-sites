# My Sites

Единая витрина авторских концептов коммерческих сайтов. На странице собраны девять самостоятельных проектов с описаниями, обложками и быстрыми переходами.

## Локальный запуск

Требуется Node.js 18 или новее.

```powershell
npm run dev
```

После запуска страница доступна по адресу `http://127.0.0.1:5191`.

Локальные проекты используют порты `5182-5190`. Витрина автоматически подставляет текущий hostname, поэтому переходы работают как через `localhost`, так и через локальный IP компьютера.

## Структура

- `index.html` - разметка витрины.
- `styles.css` - адаптивный дизайн.
- `app.js` - карточки, фильтры и проверка доступности проектов.
- `assets/` - обложки проектов.
- `scripts/server.mjs` - простой локальный сервер без внешних зависимостей.
- `scripts/build.mjs` - подготовка статической папки `dist` для Vercel.
- `vercel.json` - явная конфигурация статического деплоя.
